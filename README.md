# Thesis Results Analysis

Data analysis pipeline for a user study—loads multiple questionnaire/task CSV files, merges by common identifiers, reshapes to long format, and generates result figures.

## Overview
This repository contains a Jupyter Notebook that:
- Loads study datasets exported as CSV files
- Merges them using `Scene` and `Participant.`
- Converts the merged table to **long format** (variable/value) for plotting
- Produces comparison **boxplot figures** and saves them as PDFs

  ## Files
  - `_results.ipynb` — main notebook (data loading, merging, reshaping, plotting)
  - Output generated by the notebook:
    - `data_short_format.csv` — merged “wide” dataset (one row per Participant per Scene)
    - `data_long_format.csv` — “long” dataset (Scene, Participant, variable, value)
    - `img-results1.pdf` — boxplots for Completion Time, SUS Score, NASA-TLX Score
    - `img-results2.pdf` — boxplots for SSQ/Discomfort metrics (Nausea, Oculomotor, Disorientation, SSQ Score)
  
  ## Expected input CSV files
  Place these files in the same folder as `_results.ipynb` before running:
  - `CompletionTime.csv`
  - `Nasa.csv`
  - `SSQ.csv`
  - `SUS.csv`
  
  Each dataset must include the shared identifiers:
  - `Scene`
  - `Participant`
  
  > Note: Raw study data is not included in this repository for privacy reasons.
  
  ## How to run
  1. Install dependencies:
     - `pandas`
     - `matplotlib`
     - `seaborn`
  2. Open `_results.ipynb` in Jupyter Notebook / JupyterLab.
  3. Run all cells to generate the merged CSVs and the PDF figures.
  
  ## Outputs
  After running the notebook, you will have:
  - Merged datasets: `data_short_format.csv`, `data_long_format.csv`
  - Figures: `img-results1.pdf`, `img-results2.pdf`
